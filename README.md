This project demonstrates how to use the JavaScript Fetch API to retrieve user data from a public API and display it dynamically on a webpage.

The application fetches user information such as name, email, and address, then displays it in a structured format.

. Objective

Use JavaScript Fetch API to get data from a public API

Parse JSON response

Display user details on a webpage

Implement error handling

Add reload functionality

* API Used
https://jsonplaceholder.typicode.com/users

This API provides sample user data in JSON format.

. Technologies Used

HTML

CSS

JavaScript (Fetch API)

VS Code

Chrome Browser

. Features

Fetch user data using Fetch API

Display:

Name

Email

Address (Street & City)

Error handling using .catch()

Reload button to refetch data

Simple responsive design
